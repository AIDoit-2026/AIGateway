# 版本历史

## 2.0.1 - 2026-07-02

- 代理日志增加请求 IP 记录，便于排查下游调用来源。
- 使用日志列表和详情展示客户端 IP。
- 新增 `client_ip` 数据库字段及 SQLite / MySQL / PostgreSQL schema 产物。
- 新增 Docker 镜像构建脚本 `scripts/docker-build.sh`，默认生成当前版本标签和 `latest` 标签。

## 2.0.0 - 2026-06-28

- 项目正式更名为 AIGateway。
- 连接管理精简为 API Key only 主路径，移除客户端侧账号登录、账号令牌、签到记录和 OAuth 管理入口。
- 保留历史数据结构和部分后端兼容能力，避免旧数据因 UI 精简被自动删除。
- API Key 连接在代理失败时不再自动置为 `expired`，失败只影响通道健康和告警。
- 增加低延迟代理模式，将代理日志、成本统计、调试追踪等非必要工作转为可跳过或后台执行。
- 更新入门、配置、上游接入、管理 API、FAQ 等文档，统一引导使用 Base URL + API Key 接入。
